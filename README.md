# 🤖 AI Assistant Chatbot

A modern AI-powered chatbot built with React.js and Google Gemini API, featuring a clean mobile-first UI design.

## 🌐 Live Demo
[Click here to try the chatbot!](https://incomparable-bunny-bd3e32.netlify.app)

## ✨ Features
- 💬 Real-time AI responses powered by Google Gemini 2.5 Flash
- 📱 Mobile-first UI design (iPhone dimensions)
- ✅ Markdown rendering for formatted responses
- 💡 Suggestion chips for quick prompts
- ⏳ Typing indicator while AI is responding
- ❌ Error handling for failed requests
- 🎨 Clean and modern chat interface

## 🛠️ Tech Stack
- **Frontend:** React.js
- **AI Integration:** Google Gemini API (gemini-2.5-flash)
- **Styling:** Inline CSS with responsive design
- **Markdown:** react-markdown
- **Deployment:** Netlify
- **Version Control:** GitHub

## 🚀 Getting Started
1. Clone the repo: `git clone https://github.com/prasilla-123/ai-chatbot.git`
2. Install dependencies: `npm install`
3. Create `.env` file: `REACT_APP_GEMINI_API_KEY=your_api_key_here`
4. Run: `npm start`
